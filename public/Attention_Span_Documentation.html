01 — Project Overview

This project investigates how human attention span affects emotional brain states as measurable through scalp EEG. Subjects undergo baseline recordings (Eyes Open / Eyes Closed), watch short and long video stimuli, and are recorded once more afterward (Eyes Open After / Eyes Closed After). A multi-layer perceptron trained on four canonical emotion archetypes — Happy, Calm, Fear, Sad — is then applied to classify the emotional Valence-Arousal coordinates of all ten conditions, yielding a before-vs-after comparison of neural emotional state across four EEG frequency bands.

The pipeline runs through two parallel subject groups stored in Data/Control/ and Data/Subject/, enabling cross-group comparison. Feature extraction reuses the Mel-Frequency Cepstral Coefficient (MFCC) approach borrowed from speech processing, applied per frequency band sub-signal for each of the 19 electrode channels.

05 — Processing Pipeline

1

Data Loading

Ten CSV files per subject group are loaded as MATLAB tables using readtable(), or alternatively assigned from named workspace variables (EyesOpen, EmotionsHappy, VideoShortAS, EyesOpenAfter, etc.). Each table contains one row per 250 Hz time sample and 19 electrode columns.

EO = readtable("EO.csv");        % or: EO = EyesOpen;
EC_After = readtable("ECA.csv"); % or: EC_After = EyesClosedAfter;
2

Channel Extraction

Each electrode column is extracted from every recording table via table2array(), producing 10 stimuli × 19 channels = 190 individual time-series vectors. The column name convention is "Channel{N}_{ElectrodeName}", e.g. "Channel11_Fp1".

EO_FP1 = table2array(EO(:, "Channel11_Fp1"));
EO_FP2 = table2array(EO(:, "Channel9_Fp2"));
% ... repeated for all 19 channels × 10 stimuli
3

Frequency Band Decomposition via splitband()

Each of the 190 electrode signals is passed through an elliptic filter bank, yielding five frequency sub-bands per signal: Delta (0–4 Hz), Theta (4–8 Hz), Alpha (8–13 Hz), Beta (13–30 Hz), and Gamma (30–50 Hz), plus a wide-band DtoG (0–50 Hz) composite. This produces 950 band-filtered signals (190 × 5).

[EOGamma1,EOBeta1,EOAlpha1,EOTheta1,EODelta1] = splitband(EO_FP1);
% Note: output position maps as [pos1=Delta, pos2=Theta, pos3=Alpha,
%        pos4=Beta, pos5=Gamma] — variable names in caller are reversed
Variable Naming Note: The splitband() function signature returns outputs in the order [Delta, Theta, Alpha, Beta, Gamma, DtoG]. However, the calling code assigns them as [Gamma, Beta, Alpha, Theta, Delta] — a reversal of names. The Alpha band (position 3) is unaffected. The practical effect is that "BETA" training matrices contain Theta-filtered data and "THETA" matrices contain Beta-filtered data. The Alpha-band results remain correctly labelled.
4

MFCC Feature Extraction via melcepst()

Each of the 950 band-filtered signals is fed into melcepst(), a Mel-Frequency Cepstral Coefficient extractor adapted from speech processing. With 12 cepstral coefficients and 93 Hz floor frequency, the output for each signal is a matrix of shape [frames × 12]. This produces 950 MFCC matrices, one per band per channel per stimulus.

EOextFP1G1 = melcepst(EOGamma1, 125, 'M', 12, 14, 1024, 93);
%                         ↑fs    ↑M-norm ↑nc ↑fb ↑fftlen ↑minfrq
5

Per-Band Aggregation Across Channels

All 19 channel MFCC matrices for the same band and stimulus are stacked vertically (row-concatenation), then double-transposed — a mathematical no-op that preserves dimensions. This creates one aggregated matrix per (band, stimulus) combination. With 10 stimuli and 4 main bands, this yields 40 aggregate feature matrices (plus Gamma if used).

EOAALPHA = [EOextFP1A1; EOextFP2A2; ... EOextFP19A19];
EOAALPHA = EOAALPHA';   % transpose #1
EOAALPHA = EOAALPHA';   % transpose #2 — identity, dimensions unchanged
6

Valence-Arousal Label Encoding

For the four emotion stimuli (Happy, Calm, Fear, Sad), Valence and Arousal target values are encoded in columns 13 and 14 of each aggregate matrix. Baseline and video conditions receive neutral labels (0, 0). The four labelled emotion matrices per band are vertically stacked into a single EmotionA[BAND] training matrix.

HAALPHA(:,13) = 1;  HAALPHA(:,14) = 1;   % Happy  → (V=+1, A=+1)
CAALPHA(:,13) = 1;  CAALPHA(:,14) = -1;  % Calm   → (V=+1, A=−1)
FAALPHA(:,13) = -1; FAALPHA(:,14) = 1;   % Fear   → (V=−1, A=+1)
SAALPHA(:,13) = -1; SAALPHA(:,14) = -1;  % Sad    → (V=−1, A=−1)
EmotionAALPHA = [HAALPHA; CAALPHA; FAALPHA; SAALPHA];
7

Dataset Randomisation

The rows of each EmotionA[BAND] training matrix are randomly shuffled using randperm() to prevent ordering bias during network training. The shuffled matrix is stored as randomSig[A/B/D/T] (Alpha, Beta, Delta, Theta respectively).

[rowA, colA] = size(EmotionAALPHA);
RA = randperm(rowA);
randomSigA = EmotionAALPHA(RA, :);
8

MLP Training & Multi-Condition Testing via MLPVA()

For each frequency band, MLPVA() is called 6 times — once per non-emotion condition (EO, EC, VS, VL, EOA, ECA). The training set is always the randomised emotion matrix; the test set changes each call. This enables a coherent comparison of how emotional brain state shifts from pre-stimulus baselines through video exposure to post-stimulus recovery states. A total of 24 MLPVA calls are made across all four bands.

% Alpha band — train once, test on 6 conditions
[y,tgtestS,perAcc,out,ts,perf,time,netVA] = MLPVA(randomSigA, EOAALPHA)
[y,tgtestS,perAcc,out,ts,perf,time,netVA] = MLPVA(randomSigA, ECAALPHA)
[y,tgtestS,perAcc,out,ts,perf,time,netVA] = MLPVA(randomSigA, VSAALPHA)
[y,tgtestS,perAcc,out,ts,perf,time,netVA] = MLPVA(randomSigA, VLAALPHA)
[y,tgtestS,perAcc,out,ts,perf,time,netVA] = MLPVA(randomSigA, EOAAALPHA)
[y,tgtestS,perAcc,out,ts,perf,time,netVA] = MLPVA(randomSigA, ECAAALPHA)
[y1,y2] = plotVA2(y)

06 — Stimulus Conditions

This study uses ten distinct recording conditions, grouped into four categories: baseline resting states, emotional imagery stimuli, video stimuli of varying length, and post-stimulus recovery states. The two post-stimulus conditions (EOA, ECA) are the distinguishing feature of this attention span paradigm.

EO

Eyes Open

Pre-task resting baseline with eyes open. Subject is alert and passive. VA label (0, 0) — neutral reference state.

EC

Eyes Closed

Pre-task resting baseline with eyes closed. Typically dominated by Alpha activity. VA label (0, 0).

Happy

Happiness Induction

Positive high-arousal emotional stimulus. Target VA = (1, 1). One of four training emotions used to calibrate the MLP.

Calm

Calm Induction

Positive low-arousal emotional stimulus. Target VA = (1, −1). High valence but subdued arousal.

Sad

Sadness Induction

Negative low-arousal emotional stimulus. Target VA = (−1, −1). Low valence and low arousal.

Fear

Fear Induction

Negative high-arousal emotional stimulus. Target VA = (−1, 1). Low valence combined with high arousal.

VS

Video Short

Short-duration video stimulus. Coded as neutral (0, 0) — the emotional response is inferred by the network rather than labelled. Used as test-only condition.

VL

Video Long

Long-duration video stimulus. Neutral label (0, 0). Duration difference enables assessment of sustained attention and attentional fatigue.

EOA

Eyes Open After

Post-stimulus Eyes Open recording taken after video viewing. The neural comparison of EOA vs EO reveals residual attentional or emotional changes induced by the video. VA = (0, 0) label only.

ECA

Eyes Closed After

Post-stimulus Eyes Closed recording. Comparison with EC baseline captures how the video altered the resting neural state. Core measure of the attention span hypothesis.

07 — EEG Frequency Bands

The splitband() function decomposes each electrode signal into five frequency bands using elliptic filters designed with ellipord() and ellip(). The filters are specified at a 125 Hz Nyquist frequency (250 Hz sampling rate). Critically, the function's output order is [Delta, Theta, Alpha, Beta, Gamma, DtoG] but the calling code assigns these to variable names in reverse order.

Delta

0 – 4 Hz

Deep sleep, unconscious processes. ellipord(4/125, 5/125, 0.1, 60) — low-pass design.

Theta

4 – 8 Hz

Drowsiness, memory encoding, creativity. ellipord([4 8]/125, [2 10]/125, …)

Alpha

8 – 13 Hz

Relaxed wakefulness, eyes-closed idling. ellipord([8 13]/125, [7 14]/125, …)

Beta

13 – 30 Hz

Active concentration, cognitive engagement. ellipord([13 30]/125, [11 32]/125, …)

Gamma

30 – 50 Hz

Higher cognitive binding, perceptual processing. ellipord([30 50]/125, [28 52]/125, …)

All elliptic filters use 0.1 dB passband ripple and 60 dB stopband attenuation. The DtoG (0–50 Hz) wideband composite output is returned as the sixth output of splitband() but is not explicitly used in the feature extraction pipeline.

08 — Valence-Arousal Emotion Model

The network outputs are grounded in Russell's Circumplex Model of Affect (1980), which maps all human emotional states onto a two-dimensional plane defined by Valence (pleasant/unpleasant) and Arousal (activated/deactivated). The four training emotions occupy the four quadrants of this space.

Happy

Valence = +1 · Arousal = +1

High positive valence with high activation energy. Upper-right quadrant of the Russell model — joyful, excited, enthusiastic.

Calm

Valence = +1 · Arousal = −1

High positive valence with low activation. Lower-right quadrant — relaxed, serene, content. Strong EEG correlate with Alpha rhythms.

Fear

Valence = −1 · Arousal = +1

Negative valence with high activation. Upper-left quadrant — alarmed, tense, stressed. Strong Beta/Gamma correlates.

Sad

Valence = −1 · Arousal = −1

Negative valence with low activation. Lower-left quadrant — melancholic, depressed, lethargic. Elevated frontal Theta often observed.

Baseline conditions (EO, EC, EOA, ECA, VS, VL) are assigned neutral coordinates (0, 0) — centre of the circumplex. They serve as test-only inputs; the network classifies where they land on the VA plane relative to the four training corners. This approach is the core analytical instrument of the attention span comparison.

10 — API Reference

splitband(sig)

[Delta, Theta, Alpha, Beta, Gamma, DtoG] = splitband(sig)

Decomposes a single EEG channel time-series into six frequency sub-bands using elliptic IIR bandpass filters. All filters are designed at 125 Hz Nyquist (250 Hz sampling rate). Note that the calling code assigns output positions with reversed variable names for all bands except Alpha.

sig Input EEG channel vector [N×1]. Any single-channel time series at 250 Hz.
Delta (out 1) 0–4 Hz. Designed with ellipord(4/125, 5/125, 0.1, 60) — low-pass.
Theta (out 2) 4–8 Hz. ellipord([4 8]/125, [2 10]/125, 0.1, 60) — bandpass.
Alpha (out 3) 8–13 Hz. ellipord([8 13]/125, [7 14]/125, 0.1, 60) — bandpass.
Beta (out 4) 13–30 Hz. ellipord([13 30]/125, [11 32]/125, 0.1, 60) — bandpass.
Gamma (out 5) 30–50 Hz. ellipord([30 50]/125, [28 52]/125, 0.1, 60) — bandpass.
DtoG (out 6) 0–50 Hz wideband composite. Not used in the main pipeline feature extraction.

melcepst(sig, fs, flg, nc, p, n, inc)

cep = melcepst(sig, fs, 'M', nc, p, n, inc)

Extracts Mel-Frequency Cepstral Coefficients from a band-filtered EEG signal. Borrowed from speech processing, applied here to EEG sub-bands. Each call with the project parameters produces a [frames × 12] matrix.

sig Input band-filtered signal [N×1] — one band sub-signal from splitband().
fs = 125 Effective sampling rate. Passed as 125 (half of 250 Hz) in all calls.
flg = 'M' Magnitude spectrum mode. Computes cepstrum from log-magnitude of Mel filterbank output.
nc = 12 Number of cepstral coefficients. Feature dimensionality = 12 per frame.
p = 14 Number of Mel filterbank channels (filters) used for frequency warping.
n = 1024 FFT length in samples. Determines frequency resolution.
inc = 93 Minimum frequency bound in Hz. Filters operate from 93 Hz upward in the Mel domain.
cep (output) MFCC matrix [frames × 12]. This becomes columns 1–12 of each aggregate feature matrix.

MLPVA(randomSigT, AATHETA)

[y, tgtestS, perAcc, out, ts, perf, time, netVA] = MLPVA(randomSigT, AATHETA)

Trains a three-layer MLP on the shuffled emotion training set, then evaluates it on one test condition. Used 24 times across four bands and six non-emotion conditions.

randomSigT Training matrix [rows×14] — shuffled emotion aggregate (cols 1–12 MFCC, cols 13–14 VA labels).
AATHETA Test matrix [rows×14] — any condition aggregate (EO, EC, VS, VL, EOA, ECA per band).
y Network output [2×rows] — predicted [Valence; Arousal] for each test frame.
tgtestS Ground-truth test targets [rows×2] — VA label columns from the test matrix.
perAcc Percentage accuracy — proportion of frames where absolute error < 1.2 on both outputs.
out Absolute error matrix |tgtestS' − y|.
time / ts Training time (seconds via tic/toc) and inference time respectively.
netVA Trained network object — can be reused with sim(netVA, testData').

plotVA2(y)

[y1, y2] = plotVA2(y)

Smooths and plots the predicted Valence and Arousal trajectories over the test frames using LOESS (locally weighted regression) with a 0.1 span parameter.

y Raw network output [2×N] from MLPVA. Row 1 = Valence, Row 2 = Arousal.
y1 Smoothed Valence trajectory (red line). LOESS applied with smooth(y(1,:), 0.1, 'loess').
y2 Smoothed Arousal trajectory (blue line). Plotted in the same figure.

random(data)

randomSig = random(data)

A minimal wrapper around MATLAB's randperm() for shuffling the rows of a data matrix. In the main pipeline its logic is inlined rather than called as a function.

data Input matrix to be row-shuffled [M×N].
randomSig Row-permuted copy of data — same dimensions, rows in random order.
